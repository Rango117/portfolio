Stealth Games

A stealth game is a type of video game that tasks the player with using stealth to avoid or overcome antagonists. Games in the genre typically allow the player to remain undetected by hiding, using disguises or avoiding 
noise. Some games allow the player to choose between a stealthy approach or directly attacking antagonists, but mostly 
rewarding the player for greater levels of stealth.
